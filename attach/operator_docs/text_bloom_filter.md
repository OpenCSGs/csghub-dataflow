# 数据Bloom过滤去重 (text_bloom_filter)

## 算子功能

这是一个用来去除重复文本的工具，它使用布隆过滤器（Bloom Filter）技术来快速检测文本是否已经出现过，然后丢弃重复的文本，只保留第一次出现的文本。


## 处理逻辑

### 第一步：计算文本指纹（哈希值）

把文本转换成一个固定长度的"指纹"（哈希值）：
- "Hello world" → `5eb63bbbe01eeed093cb22bb8f5acdc3`（MD5）
- "你好世界" → `dbefd3ada018615b35588a01e216ae6e`（MD5）

不同的文本会产生不同的指纹，相同的文本会产生相同的指纹。

### 第二步：检查是否见过

查询布隆过滤器，看这个指纹是否已经存在：
- 如果存在 → 这是重复文本
- 如果不存在 → 这是新文本

### 第三步：添加到布隆过滤器

把这个指纹添加到布隆过滤器中，以便后续检测。

### 第四步：决定是否保留

- 如果是新文本 → 保留
- 如果是重复文本 → 丢弃


### 示例

**配置：**
```yaml
哈希函数: "md5"
初始容量: 100
```

**输入数据：**
```json
[
  {"text": "Hello world"},
  {"text": "你好世界"},
  {"text": "Hello world"},
  {"text": "Good morning"},
  {"text": "你好世界"}
]
```

**输出数据：**
```json
[
  {"text": "Hello world"},
  {"text": "你好世界"},
  {"text": "Good morning"}
]
```
