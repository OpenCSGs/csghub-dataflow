# 空白规范化 (whitespace_normalization_mapper)

## 算子功能

这是一个空白字符标准化器,专门将文本中各种奇怪的空白字符统一替换成标准空格,并去除首尾多余的空白。

## 处理逻辑

### 工作流程

1. **去除首尾空白**: 使用 strip() 去除文本开头和结尾的空白字符
2. **遍历每个字符**: 检查文本中的每个字符
3. **识别空白字符**: 判断字符是否属于特殊空白字符集合
4. **替换空白**: 如果是空白字符,替换为标准空格 " "
5. **保留其他字符**: 非空白字符保持不变
6. **输出结果**: 返回标准化后的文本


### 示例

**输入数据:**
```json
{"text": "   文本内容   "}
{"text": "\n\n开头有换行\n\n"}
{"text": "\t\t制表符开头\t\t"}
{"text": "正常文本"}
{"text": "　　全角空格开头　　"}
{"text": "   "}
```

**输出数据:**
```json
{"text": "文本内容"}
{"text": "开头有换行"}
{"text": "制表符开头"}
{"text": "正常文本"}
{"text": "全角空格开头"}
{"text": ""}
```
