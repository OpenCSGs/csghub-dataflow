# 平均行长度范围过滤(average_line_length_filter)

## 算子功能

这是一个用来检查文本"每行平均长度"的工具，它会计算文本中每行的平均字符数，然后根据这个平均值来决定是保留还是丢弃这段文本。


## 处理逻辑

### 第一步：分割文本

将文本按换行符分割成多行。

### 第二步：计算总字符数

统计整个文本的字符总数（包括空格、标点等）。

### 第三步：计算行数

统计文本有多少行。

### 第四步：计算平均值

平均行长度 = 总字符数 / 行数

### 第五步：判断是否保留

- 平均长度 < 最小值 → 丢弃
- 平均长度 > 最大值 → 丢弃
- 最小值 ≤ 平均长度 ≤ 最大值 → 保留


### 示例

**配置：**
```yaml
最小长度: 15
最大长度: 100
```

**输入数据：**
```json
[
  {
    "text": "人工智能是计算机科学的一个重要分支。\n它研究如何让计算机模拟人类智能。\n机器学习是实现人工智能的主要方法之一。"
  },
  {
    "text": "标题\n副标题\n章节\n段落"
  },
  {
    "text": "深度学习使用多层神经网络来学习数据的表示。\n这种方法在图像识别、语音识别等领域取得了突破性进展。"
  },
  {
    "text": "A\nB\nC"
  }
]
```

**输出数据：**
```json
[
  {
    "text": "人工智能是计算机科学的一个重要分支。\n它研究如何让计算机模拟人类智能。\n机器学习是实现人工智能的主要方法之一。"
  },
  {
    "text": "深度学习使用多层神经网络来学习数据的表示。\n这种方法在图像识别、语音识别等领域取得了突破性进展。"
  }
]
```
