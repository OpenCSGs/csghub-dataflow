# 链接地址清理(clean_links_mapper)

## 算子功能

这是一个用来清理文本中各种链接的工具，它会自动识别并移除或替换文本中的HTTP、HTTPS、FTP等各种类型的链接，包括完整URL和www开头的网址。

## 处理逻辑

### 第一步：检查文本

使用正则表达式检查文本中是否包含链接。

### 第二步：匹配链接

匹配多种格式的链接：
- 带协议的完整URL（http://、https://、ftp://等）
- www开头的网址
- 域名加路径的格式

### 第三步：替换链接

将匹配到的链接默认为空字符串，即删除

### 第四步：返回结果

返回清理后的文本。


### 示例

**输入数据：**
```json
[
  {"text": "访问 https://example.com 了解更多"},
  {"text": "官网：http://company.com"},
  {"text": "请查看 www.docs.com 的文档"},
  {"text": "下载地址：ftp://files.com/download"}
]
```

**输出数据：**
```json
[
  {"text": "访问 了解更多"},
  {"text": "官网："},
  {"text": "请查看 的文档"},
  {"text": "下载地址："}
]
```
