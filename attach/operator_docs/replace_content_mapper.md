# 文本替换 (replace_content_mapper)

## 算子功能

这是一个内容替换器,它使用正则表达式(一种强大的文本匹配规则)来找到文本中符合特定模式的内容,然后把它们替换成你想要的内容。


## 处理逻辑

算子的工作流程:

1. **准备匹配规则** - 把你提供的正则表达式编译成可以使用的规则
2. **扫描文本** - 在文本中查找符合规则的内容
3. **执行替换** - 把找到的内容替换成你指定的字符串
4. **多次替换** - 如果有多个规则,按顺序依次执行
5. **返回结果** - 返回替换后的文本


### 示例

**配置:**
```yaml
正则表达式模式列表: '●■►▼'
替换字符串: '<SPEC>'
```

**输入数据:**
```json
{
  "text": "多个●■►▼这样的特殊字符可以►▼▲▴∆吗？"
}
```

**输出数据:**
```json
{
  "text": "多个<SPEC>这样的特殊字符可以►▼▲▴∆吗？"
}
```
