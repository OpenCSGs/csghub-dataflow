# 最大行长度范围过滤 (maximum_line_length_filter)

## 算子功能

这是一个行长度检查工具,它会找出文本中最长的那一行,检查长度是否符合要求,过滤掉包含超长行或过短行的文本。

## 处理逻辑

### 第一步:分割文本

将文本按行分割:
- 使用`splitlines()`方法
- 支持各种换行符(\n、\r\n、\r)
- 得到行列表

### 第二步:计算每行长度

遍历所有行:
- 计算每行的字符数
- 包括所有字符(空格、标点等)
- 空行长度为0

### 第三步:找出最大值

找到最长的行:
- 比较所有行的长度
- 取最大值作为最大行长度
- 如果文本为空,最大行长度为0

### 第四步:判断是否保留

根据阈值判断:
- 检查最大行长度是否≥最小阈值
- 检查最大行长度是否≤最大阈值
- 两个条件都满足才保留

### 示例

**配置:**
```yaml
最小长度: 10
最大长度: 50
```

**输入数据:**
```json
{
  "text": "这是一段没有换行的超长文本内容,包含了大量的信息,但是全部挤在一行里面,这样的格式不利于阅读和处理,应该被过滤掉。"
}
```

**输出数据:**
```json
(数据被过滤,不输出,60 > 50超过最大值)
```
