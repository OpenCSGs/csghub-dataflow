# 相似去重(pipeline_magpie_zh_mapper)

## 算子功能

中文多任务对话生成器是一个智能对话数据制造工厂,能够自动创作各种类型的中文对话。它使用人工智能(AI)来生成用户和助手之间的多轮对话,涵盖11个不同的任务类型。


## 处理逻辑

1. 初始化AI客户端
2. 忽略所有输入数据(即使输入为空也能工作)
3. 遍历11个任务类型
4. 对每个任务类型,构建生成指令
5. 调用AI服务生成对话
6. 解析生成的对话文本
7. 提取用户问题和助手回答
8. 汇总所有生成的对话
9. 输出结果



### 示例

**配置文件(YAML):**
```yaml
身份验证令牌: "sk-xxxxxxxxxxxxx"
模型名称: "qwen-plus"
模型网址: "https://dashscope.aliyuncs.com/compatible-mode/v1"
```

**输入数据(JSON):**
```json
[]
```
(注意:输入数据会被完全忽略,即使为空也能正常工作)


**AI生成的对话文本:**
```
用户:长城是什么时候建造的?
助手:长城始建于春秋战国时期,距今已有2000多年的历史。
用户:长城的总长度是多少?
助手:长城的总长度约为21196公里,包括各个朝代修建的部分。
用户:哪个朝代对长城的贡献最大?
助手:明朝对长城的贡献最大,我们今天看到的长城主体大多是明朝修建的。
```
