# 数值字段范围过滤 (specified_numeric_field_filter)

## 算子功能

这是一个用来筛选数据的工具，它会检查数据中某个指定字段的数字值，然后根据这个数字是否在指定范围内来决定是保留还是丢弃这条数据。(只支持整数和浮点数类型)


## 处理逻辑

### 第一步：定位字段

根据 `field_key` 参数找到要检查的字段：
- 如果是顶层字段：直接访问（如 `star`）
- 如果是嵌套字段：用点号分隔（如 `meta.star` 或 `meta.author.age`）

### 第二步：检查是否是数字

- 尝试把字段值转换成数字
- 如果能转换成功 → 是数字，继续检查
- 如果转换失败 → 不是数字，丢弃

### 第三步：判断范围

- 如果数字 < 最小值 → 丢弃（太小了）
- 如果数字 > 最大值 → 丢弃（太大了）
- 如果数字在范围内 → 保留（刚刚好）

### 第四步：处理异常

- 如果字段不存在 → 丢弃
- 如果字段为空 → 丢弃
- 如果 field_key 为空 → 保留所有数据

### 示例

**配置：**
```yaml
字段名称: "meta.views"
最小值: 1000
最大值: 100000
```

**输入数据：**
```json
[
  {"text": "文章A", "meta": {"views": 500}},
  {"text": "文章B", "meta": {"views": 5000}},
  {"text": "文章C", "meta": {"views": 150000}}
]
```

**输出数据：**
```json
[
  {"text": "文章B", "meta": {"views": 5000}}
]
```
