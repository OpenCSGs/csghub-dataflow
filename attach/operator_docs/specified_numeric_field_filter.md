数值字段范围过滤（`specified_numeric_field_filter`）

**使用场景**
- 数值范围过滤: 根据数值字段筛选数据
- 质量控制: 过滤评分过低的数据
- 阈值过滤: 根据业务阈值筛选数据

**示例**
- 输入数据: `{"text": "...", "rating": 4.5, "views": 1000}`
- 配置: `field_key='rating', min_value=4.0, max_value=5.0`
- rating为4.5，在范围内，样本被保留

- 输入：带有数值字段（如评分、长度、点击量等）的样本。
- 输出：只保留该数值落在给定范围内的样本。
- 核心：对数值字段做区间过滤，可用于筛选高评分样本等。
