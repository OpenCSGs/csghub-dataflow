# 词级重复率范围过滤 (word_repetition_filter)

## 算子功能

这是一个词语重复过滤器,像一个重复检测器,专门检查文本中是否有太多重复的词语组合(n-gram),并根据重复比例来过滤数据。


## 处理逻辑

### 工作流程

1. **文本分词**: 将文本切分成单词列表
2. **词语精炼**: 转小写,去除特殊字符
3. **生成N-gram**: 创建所有长度为N的连续词组
4. **统计频率**: 计算每个词组出现的次数
5. **计算比例**: 重复词组数量 / 总词组数量
6. **范围判断**:
   - 比例 < 最小值 → 过滤掉(重复太少,可能是乱码)
   - 比例 > 最大值 → 过滤掉(重复太多,质量差)
   - 最小值 ≤ 比例 ≤ 最大值 → 保留
7. **输出结果**: 只保留重复比例在范围内的数据


### 示例

**配置文件(YAML):**
```yaml
语言: '中文'
分词: true
重复长度: 3
最小比例: 0.0
最大比例: 0.2  
```

**输入数据:**
```json
{"text": "机器学习算法包括监督学习、无监督学习和强化学习。"}
{"text": "今天天气很好,明天天气也很好,后天天气还是很好。"}
{"text": "Python是一种编程语言"}
{"text": "数据数据数据处理"}
{"text": "人工智能技术发展迅速,应用广泛。"}
```

**输出数据:**
```json
{"text": "机器学习算法包括监督学习、无监督学习和强化学习。"}
{"text": "Python是一种编程语言"}
{"text": "人工智能技术发展迅速,应用广泛。"}
```
