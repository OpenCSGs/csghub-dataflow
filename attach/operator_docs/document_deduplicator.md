# 文档去重MD5 (document_deduplicator)

## 算子功能

这是一个基于精确匹配的文档去重工具，它会为每个文本生成唯一的MD5指纹（哈希值），完全相同的文本会有相同的指纹，从而实现快速精确去重。

## 处理逻辑

### 第一步：预处理文本（可选）

根据配置对文本进行预处理：
- 转小写（lowercase=True）
- 移除非字母字符（ignore_non_character=True）

### 第二步：计算MD5哈希

为每个文本计算MD5哈希值，作为唯一指纹。

### 第三步：构建哈希映射

建立哈希值到文档ID的映射，识别重复文本。

### 第四步：过滤重复

遍历所有文档：
- 首次出现的哈希值：保留文档
- 已出现的哈希值：删除文档

### 第五步：返回结果

返回去重后的数据集。


### 示例

**配置：**
```yaml
小写: false
忽略非字符: false
```

**输入数据：**
```json
[
  {"text": "人工智能正在改变世界"},
  {"text": "人工智能正在改变世界"},
  {"text": "机器学习是AI的核心"},
  {"text": "Hello World"},
  {"text": "Hello  World"}

]
```

**输出数据：**
```json
[
  {"text": "人工智能正在改变世界"},
  {"text": "机器学习是AI的核心"}
  {"text": "Hello World"},
  {"text": "Hello  World"}
]
```
