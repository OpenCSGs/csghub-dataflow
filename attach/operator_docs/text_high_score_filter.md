# 打分数据筛选 (text_high_score_filter)

## 算子功能

这是一个评分范围过滤器,像一个质量检查员,专门检查数据中的评分字段是否在合理范围内。

## 处理逻辑

1. **读取评分字段**: 从数据中读取指定的评分字段(如 `text_score`)
2. **检查评分有效性**: 
   - 检查评分是否存在(不是 null)
   - 检查评分类型是否正确(必须是数字)
3. **判断评分范围**:
   - 评分 < 最小值 → 过滤掉
   - 评分 ≥ 最大值 → 过滤掉
   - 最小值 ≤ 评分 < 最大值 → 保留
4. **输出结果**: 只保留评分在有效范围内的数据

### 示例

**配置文件(YAML):**
```yaml
分数字段: 'edu_score'
最低分: 2.5
最高分: 4.5
```

**输入数据:**
```json
{"text": "机器学习是人工智能的重要分支", "edu_score": 3.8}
{"text": "今天天气不错", "edu_score": 0.5}
{"text": "Python是一种编程语言", "edu_score": 3.2}
{"text": "1+1=2", "edu_score": 4.9}
{"text": "深度学习使用神经网络", "edu_score": 4.1}
{"text": "我喜欢吃苹果", "edu_score": 0.8}
```

**输出数据:**
```json
{"text": "机器学习是人工智能的重要分支", "edu_score": 3.8}
{"text": "Python是一种编程语言", "edu_score": 3.2}
{"text": "深度学习使用神经网络", "edu_score": 4.1}
```
