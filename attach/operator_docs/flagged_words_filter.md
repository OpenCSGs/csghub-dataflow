# 标记词比例过滤 (flagged_words_filter)

## 算子功能

这是一个用来过滤不良内容的工具，它会检查文本中有多少敏感词（脏话、骂人的话、违禁词等），然后根据敏感词的比例来决定是保留还是丢弃这段文本。


## 处理逻辑

### 第一步：加载敏感词库

- 系统会从文件中加载一个敏感词列表
- 这个列表包含各种语言的敏感词（脏话、骂人的话、违禁词等）

### 第二步：分词（把文本拆成一个个词）

有两种方式：

**方式1：简单分词（默认）**
- 按空格把文本分开
- 例如："Hello world" → ["Hello", "world"]

**方式2：智能分词（需要开启）**
- 使用专业的分词工具（SentencePiece）
- 例如："我爱中国" → ["我", "爱", "中国"]
- 中文建议开启

### 第三步：清理词语

- 把所有词转成小写（"Hello" → "hello"）
- 去掉标点符号
- 可选：词语增强（把相邻的词组合起来，用于检测变体敏感词）

### 第四步：统计敏感词

- 检查每个词是否在敏感词库中
- 计算敏感词数量
- 计算敏感词比例 = 敏感词数量 / 总词数

### 第五步：判断是否保留

- 如果文本为空（0个词） → 丢弃
- 如果敏感词比例 ≤ 最大允许比例 → 保留
- 如果敏感词比例 > 最大允许比例 → 丢弃


### 示例

**配置：**
```yaml
语言: "中文"
分词: true
最大比例: 0.1
词汇增强: false
```

**输入数据：**
```json
[
  {"text": "今天天气真不错"},
  {"text": "你这个(敏感词)"},
  {"text": "我喜欢这个产品"},
  {"text": "(敏感词)"},
  {"text": "这是一个好主意"}
]
```

**输出数据：**
```json
[
  {"text": "今天天气真不错"},
  {"text": "我喜欢这个产品"},
  {"text": "这是一个好主意"}
]
```
