# 停用词比例过滤 (stopwords_filter)

## 算子功能

这是一个用来筛选文本的工具，它会检查文本中有多少停用词（常见的无意义词汇），然后根据停用词的比例来决定是保留还是丢弃这段文本。


## 处理逻辑

### 第一步：加载停用词库
- 从文件中加载停用词列表
- 不同语言有不同的停用词库

### 第二步：分词
- 把文本拆成一个个词
- 可以使用简单分词（按空格）或智能分词（SentencePiece）

### 第三步：清理词语
- 转成小写
- 去掉标点符号
- 可选：词语增强

### 第四步：统计停用词
- 检查每个词是否在停用词库中
- 计算停用词数量
- 计算停用词比例 = 停用词数量 / 总词数

### 第五步：判断是否保留
- 如果停用词比例 ≥ 最小比例 → 保留（正常文本）
- 如果停用词比例 < 最小比例 → 丢弃（异常文本）


### 示例

**配置：**
```yaml
语言: "中文"
分词: true
最小比例: 0.3
词汇增强: false
```

**停用词库（示例）：** ["的", "是", "在", "我", "你", "他", "了", "和"]

**输入数据：**
```json
[
  {"text": "我在北京的家里看书"}，
  {"text": "手机电脑平板笔记本相机耳机"}
]
```

**输出数据：**
```json
[
  {"text": "我在北京的家里看书"}
]
```
