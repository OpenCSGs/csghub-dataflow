# token数量范围过滤 (token_num_filter)

## 算子功能

这是一个Token 数量过滤器,像一个智能计数器,专门统计文本被切分成多少个 Token(词元),使用专业的 Tokenizer(分词器)来精确统计 Token 数量,比简单的字数统计更准确，并根据数量范围来过滤数据。


## 处理逻辑

### 工作流程

1. **加载 Tokenizer**: 使用指定的 Hugging Face 分词器模型
2. **文本分词**: 将文本切分成 Token 序列
3. **统计数量**: 计算 Token 的总数
4. **范围判断**:
   - Token 数 < 最小值 → 过滤掉
   - Token 数 > 最大值 → 过滤掉
   - 最小值 ≤ Token 数 ≤ 最大值 → 保留
5. **输出结果**: 只保留 Token 数量在范围内的数据


### 示例

**配置文件(YAML):**
```yaml
分词器模型: 'EleutherAI/pythia-6.9b-deduped'
最小数量: 10
最大数量: 30
```

**输入数据:**
```json
{"text": "你好"}
{"text": "今天是星期天,天气很好,我们去公园玩。"}
{"text": "人工智能正在改变世界"}
{"text": "Python是一种编程语言"}
{"text": "深度学习使用神经网络"}
```

**输出数据:**
```json
{"text": "今天是星期天,天气很好,我们去公园玩。"}
{"text": "人工智能正在改变世界"}
{"text": "深度学习使用神经网络"}
```
