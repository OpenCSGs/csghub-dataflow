# 版权信息清理 (clean_copyright_mapper)

## 算子功能

这是一个用来清理文档中版权声明、许可证信息等内容的工具，适用于 PDF→MD→JSONL 等文档流水线。算子通过用户自定义的正则表达式匹配需要删除的内容，将匹配到的部分替换为空格，并整理多余空行。

## 处理逻辑

### 第一步：解析匹配规则

将 `匹配规则` 参数解析为正则表达式列表：
- 无效正则会跳过并记录警告，不影响其他规则

### 第二步：应用正则替换

按顺序对文本应用每条正则规则，将匹配到的内容替换为空格。

### 第三步：返回结果

返回清理后的文本。


### 示例

**配置：**
```yaml
匹配规则:
Copyright\s*\([cC]\)\s*\d{4}[^.]*\.\s*All rights reserved\.
版权所有\s*\d{4}[^。]*。\s*保留所有权利。
1[3-9]\\d{9}
```

**输入数据：**
```json
{"text": "Copyright (c) 2024 Example Corp. All rights reserved.\n\n文档正文内容。"}
{"text": "版权所有 2024 某某公司。保留所有权利。\n\n这是正常段落。"}
{"text": "联系电话：13812345678，客服热线：15987654321。"}
```

**输出数据：**
```json
{"text": "文档正文内容。"}
{"text": "这是正常段落。"}
{"text": "联系电话： ，客服热线： 。"}
```
