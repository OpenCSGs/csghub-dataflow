版权信息清理（`clean_copyright_mapper`）

**使用场景**
- 代码清洗: 去除开源代码中的版权声明
- 数据预处理: 清理文档开头的版权信息
- 许可证处理: 统一处理不同格式的版权声明

**示例**
- 输入文本:
  ```
  /* 多行注释
  注释内容copyright
  */
  这是一段文本内容。
  ```
- 输出文本: `"这是一段文本内容。"`

- 输入："/* Copyright XXX 2024 */\n// some license text\nint main() {return 0;}"
- 输出："int main() {return 0;}"
- 核心：优先识别并删除开头包含 `copyright` 的 `/* ... */` 块注释；
  如果文档以一串 `//`、`#`、`--` 注释行（或空行）开头，也会整体跳过这些头部注释，保留后面的正文。
