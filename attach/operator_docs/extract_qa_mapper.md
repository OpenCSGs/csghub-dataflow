# 问答对提取 (extract_qa_mapper)

## 算子功能

这是一个智能问答生成工具，它能读懂一段文本,然后自动生成多个相关的问答对,把知识转换成对话训练格式。


## 处理逻辑

### 第一步:准备请求

构建发送给AI模型的请求:
- 系统提示词:告诉AI要做什么(提取问答对)
- 用户消息:原始文本内容
- 请求格式:OpenAI标准格式

### 第二步:调用AI模型

发送HTTP请求到AI服务:
- 使用API密钥认证
- 等待AI生成回复(最多60秒)
- 获取AI生成的问答文本

### 第三步:提取问答对

从AI回复中提取问答对:
- 使用正则表达式匹配"Human: ... Assistant: ..."格式
- 提取所有问题和答案
- 清理多余的空白字符

### 第四步:格式转换

将问答对转换为ChatML格式:
- 每个问答对包含user和assistant两条消息
- 转换为JSON数组
- 序列化为字符串

### 第五步:错误处理

如果出现问题:
- API调用失败 → 保留原文本
- 解析失败 → 保留原文本
- 未提取到问答 → 保留原文本
- 全部失败 → 任务终止


### 示例

**配置:**
```yaml
模型地址: "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions"
模型名称: "qwen-plus"
授权令牌: "sk-xxxxxxxxxxxxx"
```

**输入数据:**
```json
{
  "text": "深度学习是机器学习的一个分支,它使用多层神经网络来学习数据的表示。深度学习在图像识别、自然语言处理等领域取得了突破性进展。"
}
```

**输出数据:**
```json
{
  "text": "[{\"messages\": [{\"role\": \"user\", \"content\": \"什么是深度学习?\"}, {\"role\": \"assistant\", \"content\": \"深度学习是机器学习的一个分支,它使用多层神经网络来学习数据的表示。\"}]}, {\"messages\": [{\"role\": \"user\", \"content\": \"深度学习在哪些领域有应用?\"}, {\"role\": \"assistant\", \"content\": \"深度学习在图像识别、自然语言处理等领域取得了突破性进展。\"}]}]"
}
```
