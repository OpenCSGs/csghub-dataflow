# 多轮对话生成 (dedup_and_save_deduplicator)

## 算子功能

这是一个基于图连通性的智能去重工具，它会根据预先计算好的相似度信息，将相似的文本连接成组，每组只保留一个代表，从而实现高效去重。

## 处理逻辑

### 第一步：检查数据

检查每个样本是否包含必需的相似度信息：
- `nn_indices`：最近邻索引列表
- `nn_scores`：对应的相似度分数

### 第二步：构建相似图

创建一个图结构：
- 每个文本作为一个节点
- 相似度≥阈值的文本之间建立边

### 第三步：找连通组

使用图算法找出所有连通组（相互连接的节点集合）。

### 第四步：选择代表

每个连通组中，选择索引最小的节点作为代表，其他节点删除。

### 第五步：清理字段

将指定字段（如embedding、nn_indices等）移到stats中，减少数据体积。

### 第六步：返回结果

返回去重后的数据集。


### 示例

**配置：**
```yaml
相似阈值: 0.9
```

**输入数据：**
```json
[
  {
    "text": "机器学习是人工智能的一个分支",
    "nn_indices": [[1, 2, 3]],
    "nn_scores": [[0.95, 0.85, 0.75]]
  },
  {
    "text": "机器学习是AI的一个重要分支",
    "nn_indices": [[0, 2, 3]],
    "nn_scores": [[0.95, 0.88, 0.70]]
  },
  {
    "text": "深度学习是机器学习的子领域",
    "nn_indices": [[0, 1, 3]],
    "nn_scores": [[0.85, 0.88, 0.80]]
  },
  {
    "text": "今天天气很好",
    "nn_indices": [[0, 1, 2]],
    "nn_scores": [[0.75, 0.70, 0.80]]
  }
]
```


**输出数据：**
```json
[
  {
    "text": "机器学习是人工智能的一个分支"
  },
  {
    "text": "深度学习是机器学习的子领域"
  },
  {
    "text": "今天天气很好"
  }
]
```
