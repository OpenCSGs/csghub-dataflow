# 特殊字符清理(remove_specific_chars_mapper)

## 算子功能

这是一个专门用来删除文本中指定字符的工具,你可以告诉它要删除哪些字符,它就会把文本中所有这些字符都擦掉。

## 处理逻辑

1. **接收字符清单** - 你告诉算子要删除哪些字符
2. **构建匹配规则** - 算子根据清单构建一个查找规则
3. **扫描文本** - 逐个检查文本中的每个字符
4. **删除匹配字符** - 如果字符在清单中,就删除它
5. **返回结果** - 返回删除后的干净文本

### 示例

**配置文件(YAML)**:
```yaml
需要移除的字符: '◆●■►▼▲▴∆▻▷❖♡□'
```

**输入数据**:
```json
{
  "text": "多个●■►▼这样的特殊字符可以►▼▲▴∆吗？"
}
```

**输出数据**:
```json
{
  "text": "多个这样的特殊字符可以吗？"
}
```
