# 字符级重复率范围过滤 (character_repetition_filter)

## 算子功能

这是一个用来检测文本中"字符片段重复"的工具，它会查找文本中重复出现的字符片段（n-gram），计算重复的比例，然后根据这个比例来决定是保留还是丢弃这段文本。


## 处理逻辑

### 第一步：提取n-gram

从文本中提取所有长度为n的字符片段。

### 第二步：统计频率

统计每个n-gram出现的次数。

### 第三步：识别重复

找出出现次数大于1的n-gram（重复的片段）。

### 第四步：计算比例

计算重复n-gram占所有n-gram的比例。

### 第五步：判断是否保留

- 比例 < 最小值 → 丢弃
- 比例 > 最大值 → 丢弃
- 最小值 ≤ 比例 ≤ 最大值 → 保留

### 示例

**配置：**
```yaml
重复长度: 5
最小比例: 0.0
最大比例: 0.5
```

**输入数据：**
```json
[
  {"text": "今天天气很好，今天天气很好，今天天气很好"},
  {"text": "人工智能是计算机科学的一个重要分支"},
  {"text": "哈哈哈哈哈哈哈哈哈哈哈哈哈哈"},
  {"text": "机器学习包括监督学习和无监督学习"}
]
```

**输出数据：**
```json
[
  {"text": "人工智能是计算机科学的一个重要分支"},
  {"text": "机器学习包括监督学习和无监督学习"}
]
```
