# 排序范围选择器(range_specified_field_selector)

## 算子功能

范围字段选择器是一个智能排名筛选工具,能够根据某个字段的分数对样本进行排序,然后选择排名在指定范围内的样本。


## 处理逻辑

1. 检查数据集大小和参数设置
2. 提取每个样本的指定字段值
3. 按字段值从小到大排序
4. 计算下界排名(lower_bound)
5. 计算上界排名(upper_bound)
6. 选择排名在[下界,上界)范围内的样本
7. 返回选择的样本
8. 记录统计信息


### 示例

**配置文件(YAML):**
```yaml
字段名称: score
下百分位: 0.25
上百分位: 0.75
下限排名: 2
上限排名: 7
```

**输入数据:**
```json
[
  {"text": "样本1", "score": 10},
  {"text": "样本2", "score": 20},
  {"text": "样本3", "score": 30},
  {"text": "样本4", "score": 40},
  {"text": "样本5", "score": 50},
  {"text": "样本6", "score": 60},
  {"text": "样本7", "score": 70},
  {"text": "样本8", "score": 80},
  {"text": "样本9", "score": 90},
  {"text": "样本10", "score": 100}
]
```


**输出数据:**
```json
[
  {"text": "样本3", "score": 30},
  {"text": "样本4", "score": 40},
  {"text": "样本5", "score": 50},
  {"text": "样本6", "score": 60},
  {"text": "样本7", "score": 70}
]
```
