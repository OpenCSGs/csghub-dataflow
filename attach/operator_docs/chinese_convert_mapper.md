# 汉字转换 (chinese_convert_mapper)

## 算子功能

这是一个用来转换中文字体的工具，它可以在简体中文、繁体中文和日文汉字之间进行转换，支持多种地区标准（大陆、台湾、香港）。

## 处理逻辑

### 第一步：选择转换模式

根据需求选择合适的转换模式（如 简体转繁体、繁体转简体 等）。

### 第二步：读取文本

获取需要转换的文本内容。

### 第三步：执行转换

使用OpenCC库进行字体转换。

### 第四步：替换原文本

将转换后的文本替换原来的文本。

### 示例

**配置：**
```yaml
转换模式: 简体转繁体
```

**输入数据：**
```json
[
  {"text": "我爱学习中文"},
  {"text": "这是一个简单的例子"},
  {"text": "机器学习很有趣"},
  {"text": "Hello World"}
]
```

**输出数据：**
```json
[
  {"text": "我愛學習中文"},
  {"text": "這是一個簡單的例子"},
  {"text": "機器學習很有趣"},
  {"text": "Hello World"}
]
```
