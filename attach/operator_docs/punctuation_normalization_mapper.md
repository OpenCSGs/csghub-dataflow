# Unicode标点规范化(punctuation_normalization_mapper)

## 算子功能

标点符号标准化器是一个文本标点转换工具,能够将各种语言的标点符号(中文、法文、德文等)统一转换成英文标点符号。


## 处理逻辑

1. 准备一个标点符号转换对照表(54种标点)
2. 读取文本内容
3. 逐个检查每个字符
4. 如果是需要转换的标点,替换成英文标点
5. 如果不是标点或不在转换表中,保持原样
6. 输出转换后的文本
7. 记录转换统计信息


### 示例1

**输入数据(JSON):**
```json
{
  "text": "你好，世界！这是一个测试。",
  "meta": {"source": "chinese_text"}
}
```

**输出数据(JSON):**
```json
{
  "text": "你好,世界!这是一个测试.",
  "meta": {"source": "chinese_text"}
}
```
**说明:** 中文标点符号被转换为对应的英文标点,中文文字保持不变。
