# 移除长字符串(remove_long_words_mapper)

## 算子功能

这是一个专门用来过滤文本中异常长度单词的工具,它能自动识别并删除过短或过长的单词,只保留长度合理的单词。


## 处理逻辑

算子的工作流程分为四个步骤:

1. **文本分割** - 按空格、制表符、换行符把文本分割成一个个单词
2. **长度检查** - 检查每个单词的长度是否在设定的范围内
3. **过滤单词** - 删除长度不符合要求的单词
4. **文本重组** - 把保留的单词重新组合成文本,保持原有的格式

**特殊处理**: 算子会用两种方式判断单词长度:
- 方式1: 直接计算单词的字符数
- 方式2: 先去掉单词两端的标点符号,再计算字符数

只要满足任何一种方式,单词就会被保留。


### 示例

**配置文件(YAML)**:
```yaml
最小长度: 1
最大长度: 10
```

**输入数据(JSON)**:
```json
{
  "text": "Hello 世界 Test 测试 verylongword"
}
```

**输出数据(JSON)**:
```json
{
  "text": "Hello 世界 Test 测试"
}
```
