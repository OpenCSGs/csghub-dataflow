# HTML代码清理 (clean_html_mapper)

## 算子功能

这是一个用来清理文本中HTML代码的工具，它会自动识别并移除文本中的HTML标签，只保留纯文本内容，同时对列表标签进行特殊处理以保持可读性。


## 处理逻辑

### 第一步：特殊处理列表标签

将列表标签转换为可读格式：
- `<li>` → `\n*`（换行加星号）
- `</li>` → 删除
- `<ol>` → `\n*`（换行加星号）
- `</ol>` → 删除

### 第二步：解析HTML

使用HTML解析器解析整个HTML代码。

### 第三步：提取纯文本

从解析后的HTML中提取所有文本内容，移除所有标签。

### 第四步：返回结果

返回清理后的纯文本。


### 示例

**输入数据：**
```json
[
  {"text": "<p>这是一个<strong>重要</strong>的段落。</p>"},
  {"text": "<div>欢迎访问<a href='https://example.com'>我们的网站</a></div>"},
  {"text": "<h1>标题</h1><p>内容在这里。</p>"},
  {"text": "<ul><li>第一项</li><li>第二项</li><li>第三项</li></ul>"}
]
```

**输出数据：**
```json
[
  {"text": "这是一个重要的段落。"},
  {"text": "欢迎访问我们的网站"},
  {"text": "标题\n内容在这里。"},
  {"text": "\n*第一项\n*第二项\n*第三项"}
]
```
