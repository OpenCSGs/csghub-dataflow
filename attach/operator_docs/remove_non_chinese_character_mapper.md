# 移除非中文字符(remove_non_chinese_character_mapper)

## 算子功能

这是一个专门用来清理中文文本的工具,它能自动识别并删除文本中的非中文字符,可以选择性地保留英文字母、数字和标点符号。


## 处理逻辑

1. **构建过滤规则** - 根据参数设置,确定要保留哪些类型的字符
2. **扫描文本** - 逐个检查文本中的每个字符
3. **判断保留** - 判断字符是否属于保留范围(中文、字母、数字、标点)
4. **删除字符** - 删除不在保留范围内的所有字符
5. **返回结果** - 返回清理后的文本

### 示例

**配置文件(YAML)**:
```yaml
保留字母: false
保留数字: false
保留标点符号: false
```

**输入数据(JSON)**:
```json
{
  "text": "👊    所有的非汉字a44sh都12@46h会被*&……*qb^4525去掉"
}
```

**输出数据(JSON)**:
```json
{
  "text": "所有的非汉字都会被去掉"
}
```
