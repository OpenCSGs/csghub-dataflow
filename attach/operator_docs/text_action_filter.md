# 动作文本过滤 (text_action_filter) 

## 算子功能

这是一个文本动作过滤器,它会调用AI大模型(比如通义千问)来分析文本,数一数文本里有多少个动作词(动词),然后根据你设定的最少动作词数量来决定是否保留这段文本。

## 处理逻辑

算子的工作流程:

1. **准备文本** - 清理文本中的特殊标记
2. **调用AI** - 把文本发送给大语言模型(LLM)
3. **AI分析** - AI识别文本中的所有动作词
4. **统计数量** - AI返回动作词的数量
5. **判断过滤** - 如果动作词数量 >= 最小要求,保留;否则过滤


### 示例

**配置:**
```yaml
base_url: 'https://dashscope.aliyuncs.com/compatible-mode/v1'
model: 'qwen-max'
api_key: 'your-api-key'
min_action_num: 1
```

**输入数据:**
```json
{
  "text": "我今天去公园跑步，然后回家吃饭。"
}
```

**输出数据:**
```json
{
  "text": "我今天去公园跑步，然后回家吃饭。",
  "stats": {
    "num_action": 4
  }
}
```
