# 前部样本的选择器 (topk_specified_field_selector)

## 算子功能

这是一个Top-K 选择器,像一个排行榜筛选器,根据指定字段的数值大小进行排序,然后选出排名最靠前的 K 个数据。


## 处理逻辑

### 工作流程

1. **读取字段值**: 从每条数据中提取指定字段的数值
2. **排序**: 根据数值大小进行排序(升序或降序)
3. **计算选择数量**:
   - 如果设置了 topk,选择前 K 个
   - 如果设置了 top_ratio,选择前 X% 个
   - 如果两者都设置,取较小的数量
4. **选择数据**: 使用堆算法高效选出 Top-K 数据
5. **输出结果**: 返回选中的数据集


### 示例

**配置**:
```yaml
字段名称: 'score'
前n%比率: 0
前k个: 3
反向排序: true
```

**输入数据**:
```json
[
  {"text": "文章A", "score": 8.5},
  {"text": "文章B", "score": 9.2},
  {"text": "文章C", "score": 7.8},
  {"text": "文章D", "score": 9.5},
  {"text": "文章E", "score": 8.0}
]
```

**输出数据**:
```json
[
  {"text": "文章D", "score": 9.5},
  {"text": "文章B", "score": 9.2},
  {"text": "文章A", "score": 8.5}
]
```
