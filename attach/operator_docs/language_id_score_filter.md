# 特定语言置信度过滤 (language_id_score_filter)

## 算子功能

这是一个智能语言识别工具,它能自动识别文本是什么语言,并根据识别的准确度来筛选数据,保留你想要的语言。


## 处理逻辑

### 第一步:加载模型

加载fastText语言识别模型:
- 预训练模型,支持170+种语言
- 模型大小约130MB
- 首次使用会自动下载

### 第二步:文本预处理

清理文本以提高识别准确度:
- 转换为小写
- 替换换行符为空格
- 保持文本连续性

### 第三步:语言识别

调用模型识别语言:
- 输入预处理后的文本
- 模型返回语言代码(如'zh'、'en')
- 同时返回置信度分数(0-1之间)

### 第四步:提取结果

处理模型输出:
- 去除标签前缀'__label__'
- 提取纯语言代码
- 记录置信度分数

### 第五步:判断是否保留

根据配置决定是否保留:

**情况1:指定了目标语言**
- 检查识别的语言是否在目标列表中
- 检查置信度是否≥最小阈值
- 两个条件都满足才保留

**情况2:未指定目标语言**
- 只检查置信度是否≥最小阈值
- 不限制语言类型


### 示例

**配置:**
```yaml
语言: "中文"
最小分数: 0.8
```

**输入数据:**
```json
{
  {"text": "这是一段中文文本,介绍了人工智能的发展历史。"},
  {"text": "This is an English sentence about artificial intelligence."}
}
```

**输出数据:**
```json
{
  "text": "这是一段中文文本,介绍了人工智能的发展历史。",
  "stats": {
    "lang": "zh",
    "lang_score": 0.95
  }
}
```
