# 风格数据合成 (make_cosmopedia_mapper)

## 算子功能

这是一个智能教程生成工具,它能把简短的网页摘录变成详细的WikiHow风格教程,自动补充步骤说明和背景知识。

## 处理逻辑

### 第一步:字段标准化

统一数据格式:
- 如果有`content`字段但没有`text` → 重命名为`text`
- 如果有`md`字段但没有`text` → 重命名为`text`
- 确保所有数据都有`text`字段

### 第二步:构建网页摘录

组合标题和内容:
- 提取标题:`sample.get('title', '')`
- 组合格式:`标题 + '\n' + 文本`
- 长度检查:超过最大长度则截断
- 截断格式:`前800字符 + "......"`

### 第三步:准备API请求

构建对话消息:
- 系统角色:"你是一个乐于助人的助手"
- 用户消息:使用提示模板,插入网页摘录
- 提示要求:生成WikiHow风格的详细教程

### 第四步:调用AI模型

发送请求到API:
- 使用配置的模型URL和认证令牌
- 流式接收AI生成的内容
- 拼接所有内容片段

### 第五步:保存结果

存储生成的教程:
- 成功:将教程保存到`sample['data']`
- 失败:将空字符串保存到`sample['data']`
- 保留原始数据不变

### 示例1:狗狗训练教程

**配置:**
```yaml
网页文本最大长度: 800
model_url: "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions"
model: "qwen-plus"
auth_token: "sk-xxxxxxxxxxxxx"
content : "网页摘录：“{web_text}”。以 WikiHow 的风格写一篇长而非常详细的教程，教程与此网页摘录有相关性。教程中需要包括对每个步骤的深入解释以及它如何帮助实现预期结果。你可以自由补充其他相关知识。确保清晰性和实用性，让读者能够轻松遵循教程完成任务。内容中不应包含广告或涉及隐私的信息。不要使用图像。请直接开始撰写教程。"
```

**输入数据:**
```json
{
  "title": "如何训练狗狗坐下",
  "text": "训练狗狗坐下是最基本的命令之一。首先准备一些零食作为奖励。"
}
```
