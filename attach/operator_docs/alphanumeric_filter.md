# 字母/数字占比过滤 (alphanumeric_filter)

## 算子功能

这是一个用来检查文本中"有效字符"比例的工具，它会计算文本中字母和数字占总内容的比例，然后根据这个比例来决定是保留还是丢弃这段文本。


## 处理逻辑

### 第一步：计算比例

**模式1（按字符）：**
1. 统计文本中所有字母和数字的数量
2. 统计文本总字符数
3. 计算比例 = 字母数字数量 / 总字符数

**模式2（按词元）：**
1. 统计文本中所有字母的数量（不包括数字）
2. 使用分词器统计词元数量
3. 计算比例 = 字母数量 / 词元数量

### 第二步：判断是否保留

- 比例 < 最小值 → 丢弃（内容质量太低）
- 比例 > 最大值 → 丢弃（超出范围）
- 最小值 ≤ 比例 ≤ 最大值 → 保留（符合要求）


### 示例

**配置：**
```yaml
分词: false
最小比例: 0.5
最大比例: 1.0
```

**输入数据：**
```json
[
  {"text": "Hello World"},
  {"text": "你好世界"},
  {"text": "!!!@@@###"},
  {"text": "Test测试123"}
]
```

**输出数据：**
```json
[
  {"text": "Hello World"},
  {"text": "Test测试123"}
]
```
