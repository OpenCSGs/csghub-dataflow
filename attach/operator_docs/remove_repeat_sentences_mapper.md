重复句子移除（`remove_repeat_sentences_mapper`）

**使用场景**
- 文本去重: 删除文档中重复出现的句子
- 数据清洗: 去除冗余内容
- 质量提升: 提高文本的信息密度

**示例**
- 输入文本:
  ```
  今天天气真不错，阳光明媚，适合出去散步。
  小明说："今天天气真不错，我们去海边吧。"
  小红回答说:"好主意！"
  但是，小李觉得："今天天气真不错，我们去爬山吧。"
  今天天气真不错，阳光明媚，适合出去散步。
  昨天下了一整天的雨，今天终于放晴了。
  昨天下了一整天的雨，今天终于放晴了。
  ```
- 配置: `lowercase=False, ignore_special_character=True, min_repeat_sentence_length=2`
- 输出文本:
  ```
  今天天气真不错，阳光明媚，适合出去散步。
  小明说："今天天气真不错，我们去海边吧。"
  小红回答说:"好主意！"
  但是，小李觉得："今天天气真不错，我们去爬山吧。"
  昨天下了一整天的雨，今天终于放晴了。
  ```
