# 句子去重(remove_repeat_sentences_mapper)

## 算子功能

这是一个专门用来删除文本中重复句子的工具,它能自动识别并删除重复出现的句子,只保留第一次出现的句子。


## 处理逻辑

算子的工作流程分为五个步骤:

1. **分行处理** - 按换行符把文本分成多行
2. **句子分割** - 在每行内,按标点符号(句号、问号、感叹号等)把文本分成一个个句子
3. **句子标准化** - 根据参数设置,可能会转换大小写、去除特殊字符
4. **去重判断** - 用一个"记忆本"(HashSet)记录已经出现过的句子,如果句子已经出现过就删除
5. **文本重组** - 把保留的句子重新组合成文本


### 示例

**配置文件(YAML)**:
```yaml
小写: true
忽略特殊字符: true
最小重复句子长度: 2
```

**输入数据(JSON)**:
```json
{
  "text": "Hello World. hello world. HELLO WORLD."
}
```

**输出数据(JSON)**:
```json
{
  "text": "Hello World."
}
```
