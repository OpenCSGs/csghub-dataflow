# 文本长度范围过滤 (text_length_filter)

## 算子功能

这是一个用来筛选文本的工具，它会数一数文本有多少个字符（包括字母、汉字、标点、空格等所有内容），然后根据你设定的最短长度和最长长度来决定是保留还是丢弃这段文本。

## 处理逻辑


### 第一步：数字符数

- 直接数文本里有多少个字符
- 包括：字母、汉字、数字、标点符号、空格、换行符等所有内容
- 例如："Hello world" → 11个字符（包括中间的空格）
- 例如："你好世界" → 4个字符

### 第二步：判断是否保留

- 如果字符数 < 最短长度 → 丢弃（太短了）
- 如果字符数 > 最长长度 → 丢弃（太长了）
- 如果字符数在范围内 → 保留（刚刚好）

### 第三步：记录结果

- 把字符数和判断结果保存下来
- 方便后续查看为什么某个文本被保留或丢弃


### 示例

**配置：**
```yaml
最小长度: 5
最大长度: 100
```

**输入数据：**
```json
[
  {"text": "你好"},
  {"text": "今天天气真不错"},
  {"text": "嗨"},
  {"text": "这是一个测试文本内容"}
]
```

**输出数据：**
```json
[
  {"text": "今天天气真不错"},
  {"text": "这是一个测试文本内容"}
]
```
