# 表格内容清理 (remove_table_text_mapper)

## 算子功能

这是一个表格文本移除器,它会在文本中寻找那些看起来像表格的内容(有行有列、整整齐齐排列的数据),然后把它们删掉,只保留普通的文字段落。

## 处理逻辑

1. **扫描文本** - 从头到尾检查文本内容
2. **识别表格** - 找出那些用空格或Tab分隔、有多行多列的内容
3. **检查列数** - 看表格的列数是否在设定范围内(比如2-20列)
4. **检查行数** - 确认至少有2行数据(单行不算表格)
5. **删除表格** - 把符合条件的表格全部删掉
6. **保留其他** - 表格前后的普通文字都保留


### 示例

**配置:**
```yaml
最小列数: 2
最大列数: 20
```

**输入数据:**
```json
{
  "text": "这是一个财务报告:\n编号 分行 营运资金1 营运资金2 营运资金3\n① 北京分行 495000000 200000000 295000000\n② 大连分行 440000000 100000000 340000000\n③ 重庆分行 500000000 100000000 400000000\n报告结束。"
}
```

**输出数据:**
```json
{
  "text": "这是一个财务报告:\n报告结束。"
}
```
