# 关键字过滤 (multi_keyword_filter)

## 算子功能

这是一个用来过滤包含特定关键词的文本的工具，它会检查文本中是否包含任何指定的关键词，如果包含就丢弃，不包含就保留。


## 处理逻辑

### 第一步：获取文本内容

从数据中读取要检查的文本

### 第二步：检查是否包含关键词

遍历所有关键词，检查文本中是否包含：
- 如果找到任何一个关键词 → 标记为"包含"
- 如果没有找到任何关键词 → 标记为"不包含"

### 第三步：决定是否保留

- 如果包含关键词 → 丢弃
- 如果不包含关键词 → 保留


### 示例

**配置：**
```yaml
关键字: "广告","垃圾"
区分大小写: false
```

**输入数据：**
```json
[
  {"text": "这是一篇正常的文章"},
  {"text": "这是一条广告信息"},
  {"text": "这是垃圾内容"},
  {"text": "This is spam"},
]
```

**输出数据：**
```json
[
  {"text": "这是一篇正常的文章"},
  {"text": "This is spam"},
]
```
