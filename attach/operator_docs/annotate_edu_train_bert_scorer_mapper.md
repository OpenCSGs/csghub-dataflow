# 教学评估打分 (annotate_edu_train_bert_scorer_mapper)

## 算子功能

这是一个用来评估文本"教育价值"的工具，它会将你的文本内容与一个参考问题进行对比，计算两者的相似度，然后给出一个0-5分的教育相关度评分。

## 处理逻辑

### 第一步：准备参考问题

设置一个教育相关的参考问题，用来衡量文本的教育价值。

### 第二步：获取文本向量

使用AI模型将参考问题和文本内容都转换成数字向量（embedding）。

### 第三步：计算相似度

计算两个向量之间的余弦相似度（cosine similarity）。

### 第四步：映射到分数

根据相似度范围，将其映射到0-5分的评分。

### 第五步：添加评分字段

在原始数据中添加一个新字段 `text_score`，存储计算出的分数。


### 示例

**配置：**
```yaml
身份验证令牌: "your-api-token"
模型网址: "https://dashscope.aliyuncs.com/compatible-mode/v1"
模型名称: "text-embedding-v4"
尺寸: 1024
查询文本: "什么是深度学习？"
```

**输入数据：**
```json
[
  {"text": "深度学习是机器学习的一个分支，它使用多层神经网络来学习数据的表示。"},
  {"text": "今天天气真好，适合出去散步。"},
  {"text": "神经网络是深度学习的基础，包含输入层、隐藏层和输出层。"},
  {"text": "我喜欢吃苹果。"}
]
```

**输出数据：**
```json
[
  {
    "text": "深度学习是机器学习的一个分支，它使用多层神经网络来学习数据的表示。",
    "text_score": 4.25
  },
  {
    "text": "今天天气真好，适合出去散步。",
    "text_score": 0.40
  },
  {
    "text": "神经网络是深度学习的基础，包含输入层、隐藏层和输出层。",
    "text_score": 3.90
  },
  {
    "text": "我喜欢吃苹果。",
    "text_score": 0.27
  }
]
```
