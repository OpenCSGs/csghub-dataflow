# 邮箱后缀清理 (clean_email_mapper)

## 算子功能

这是一个用来清理文本中邮箱地址的工具，它会自动找到文本中的所有邮箱地址，然后将它们删除或替换成其他内容。


## 处理逻辑

### 第一步：搜索邮箱

使用正则表达式在文本中搜索邮箱地址。

**默认匹配规则：**
- 用户名：字母、数字、点、减号、加号、下划线
- @ 符号
- 域名：小写字母、数字、点、减号、加号、下划线
- 点号
- 后缀：小写字母

### 第二步：判断是否存在

- 找到邮箱 → 执行替换
- 没找到邮箱 → 保持原样

### 第三步：执行替换

将找到的邮箱地址替换为指定内容（默认是删除）。


### 示例

**输入数据：**
```json
[
  {"text": "请联系我的邮箱：zhang@example.com"},
  {"text": "发送到 admin@company.cn 获取更多信息"},
  {"text": "Email地址：support@service.org"},
  {"text": "没有邮箱地址"}
]
```

**输出数据：**
```json
[
  {"text": "请联系我的邮箱："},
  {"text": "发送到  获取更多信息"},
  {"text": "Email地址："},
  {"text": "没有邮箱地址"}
]
```
