# IP地址清理 (clean_ip_mapper)

## 算子功能

这是一个用来清理文本中IP地址的工具，它会自动识别并移除或替换文本中的IPv4和IPv6地址，保护隐私或清理敏感信息。


## 处理逻辑

### 第一步：检查文本

使用正则表达式检查文本中是否包含IP地址。

### 第二步：匹配IP地址

- 匹配IPv4地址（如：192.168.1.1）
- 匹配IPv6地址（如：2001:0db8:85a3:0000:0000:8a2e:0370:7334）

### 第三步：替换IP地址

将匹配到的IP地址默认为空字符串，即删除

### 第四步：返回结果

返回清理后的文本。


### 示例

**输入数据：**
```json
[
  {"text": "服务器地址是 192.168.1.1"},
  {"text": "请访问 10.0.0.1 进行配置"},
  {"text": "DNS设置为 8.8.8.8"},
  {"text": "网关地址：172.16.0.1"}
]
```

**输出数据：**
```json
[
  {"text": "服务器地址是 "},
  {"text": "请访问 进行配置"},
  {"text": "DNS设置为 "},
  {"text": "网关地址："}
]
```
