# 特殊字符比例过滤 (special_characters_filter)

## 算子功能

这是一个用来筛选文本的工具，它会数一数文本中有多少特殊字符（标点符号、数字、表情符号等），然后根据特殊字符的比例来决定是保留还是丢弃这段文本。


## 处理逻辑

### 第一步：统计特殊字符
- 遍历文本中的每个字符
- 检查是否是特殊字符
- 统计特殊字符的数量

### 第二步：计算比例
```
特殊字符比例 = 特殊字符数量 / 文本总字符数
```

### 第三步：判断是否保留
- 如果比例 < 最小比例 → 丢弃
- 如果比例 > 最大比例 → 丢弃
- 如果比例在范围内 → 保留


### 示例1

**配置：**
```yaml
最小比例: 0.0
最大比例: 0.25
```

**输入数据：**
```json
[
  {"text": "This is a test"},
  {"text": "!!!"},
  {"text": "正常的中文文本"},
  {"text": "Mix 中英文 text"},
  {"text": "😊😊😊😊😊"}
]
```

**输出数据：**
```json
[
  {"text": "This is a test"},
  {"text": "正常的中文文本"},
  {"text": "Mix 中英文 text"}
]
```
