# 单词数量范围过滤 (words_num_filter)

## 算子功能

这是一个用来筛选文本的工具，它会数一数文本里有多少个词，然后根据你设定的最少词数和最多词数来决定是保留还是丢弃这段文本。(中文不开启分词会出错！所有文本都被当成1个词)


## 处理逻辑

### 第一步：分词（把文本拆成一个个词）

有两种方式：

**方式1：简单分词（默认）**
- 按空格把文本分开
- 例如："Hello world" → ["Hello", "world"]（2个词）
- 适合英文，但中文不适合（因为中文没有空格）

**方式2：智能分词（需要开启）**
- 使用专业的分词工具（SentencePiece）
- 例如："我爱中国" → ["我", "爱", "中国"]（3个词）
- 适合中文和其他语言

### 第二步：清理词语（去掉标点符号等）

- 把标点符号、特殊字符去掉
- 例如："Hello!" → "Hello"
- 例如："!!!" → 删除（只有标点，没有实际内容）

### 第三步：数词数

- 数一数清理后还剩多少个词
- 例如：["Hello", "world", "test"] → 3个词

### 第四步：判断是否保留

- 如果词数 < 最少词数 → 丢弃（太短了）
- 如果词数 > 最多词数 → 丢弃（太长了）
- 如果词数在范围内 → 保留（刚刚好）

### 示例1

**配置：**
```yaml
语言: "中文"
分词: true
最小数量: 5
最大数量: 100
```

**输入数据：**
```json
[
  {"text": "我爱中国"},
  {"text": "根据算子使用使用使用使用安装方案确定"},
  {"text": "今天天气真不错，适合出去玩"},
  {"text": "这"}
]
```

**输出数据：**
```json
[
  {"text": "根据算子使用使用使用使用安装方案确定"},
  {"text": "今天天气真不错，适合出去玩"}
]
```
