# 字段信息过滤 (specified_field_filter) 

## 算子功能

这是一个指定字段过滤器,就像一个专门检查标签的筛选员。它会检查数据中某个字段的值,只保留字段值在你指定的"允许列表"中的数据,其他的都过滤掉。

## 处理逻辑

算子的工作流程:

1. **检查配置** - 看看是否设置了字段名和允许列表
2. **访问字段** - 找到数据中指定的字段(支持多层嵌套,如`meta.suffix`)
3. **获取字段值** - 读取字段的值
4. **检查是否在列表中** - 看字段值是否在允许列表中
5. **做出决定** - 在列表中就保留,不在就过滤

### 示例

**配置:**
```yaml
字段名称: 'category'
目标值: ['技术', '科学']
```

**输入数据:**
```json
{"text": "Python编程入门", "category": "技术"}
{"text": "今天天气很好", "category": "生活"}
{"text": "量子物理学基础", "category": "科学"}
{"text": "美食推荐", "category": "美食"}
{"text": "机器学习算法", "category": "技术"}
{"text": "旅游攻略", "category": "旅游"}
```

**输出数据:**
```json
{"text": "Python编程入门", "category": "技术"}
{"text": "量子物理学基础", "category": "科学"}
{"text": "机器学习算法", "category": "技术"}
```
